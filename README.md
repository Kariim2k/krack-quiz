# 🎯 Krack Quiz

Krack Quiz is a TypeScript-based quiz application built with Vite. Users can create accounts, log in, answer timed quiz questions, track their score, and install the app as a Progressive Web App (PWA).

## 🚀 Live Demo

https://krack-quiz.vercel.app/

---

## ✨ Features

- User signup and login
- Multiple user accounts stored with Local Storage
- Timed quiz system
- Pause timer on logout
- Resume timer after login
- Score tracking
- Progress indicator
- Progress bar
- Restart quiz functionality
- Show / Hide password
- Authentication validation
- Custom Krack Quiz branding and icon
- Installable PWA support
